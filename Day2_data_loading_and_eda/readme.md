# 🐼 Day 2 — EDA (Titanic Dataset)

## 📊 Dataset Overview
The dataset contains passenger information such as age, gender, ticket class, and survival status.

---

## 🔍 Key Findings

1. Survival Rate  
Most passengers did not survive (~62%), while only ~38% survived.

2. Gender Distribution  
Male passengers were significantly higher than female passengers.

3. Missing Data  
The "Cabin" column has the highest number of missing values.
